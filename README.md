# Agri Export System

## Overview

A Node.js + Express based web application to manage agricultural export data.

## Features

- Add and manage export records
- Serve static frontend (HTML, CSS, JS)
- REST API endpoints for data handling

## Installation

```bash
git clone https://github.com/shivachaudhary7701-sketch/agri-export-system.git
cd agri-export-system
npm install
npm start
```
